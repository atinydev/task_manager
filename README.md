# Task Manager Flutter

A simple application to manage tasks with Api connection.

You can create, edit, and delete tasks.

### Task:
 - title -> Text
 - is_completed -> 0 - 1
 - due_date -> YYYY-MM-DD
 - comments -> Text
 - description -> Text
 - tags -> Text

## Run
```
flutter pub get
flutter run
```
